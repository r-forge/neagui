Network Enrichment Analysis GUI
neaGUI Input

The main input of neaGUI is a list of altered gene sets (AGSs) which is usually a list of differentially expressed genes and gene network links. 
The gene network input can be a vector of gene pairs or a list representing the network link. For the vector case, each element has a combined name of two gene symbols with space separation.
